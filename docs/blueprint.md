# **App Name**: Through Hardik's Eye

## Core Features:

- Home: Display a captivating intro and brief bio.
- Portfolio: Showcase diverse photography (portraits, events, fashion, concerts, street, weddings) in distinct sections with unique layouts and animations on each photo.
- Contacts: Provide a contact form and information for inquiries.
- Testimonials: Display testimonials from past clients.
- AI Portfolio Curator Tool: AI tool selects which photographs should be in the main portfolio based on image aesthetics, customer sentiment scores, and seasonal demand.

## Style Guidelines:

- Primary color: Desaturated blue (#87CEEB) for tranquility and reflection, echoing Picasso's Blue Period.
- Background color: Very light desaturated blue (#F0F8FF), almost white, for a subtle backdrop.
- Accent color: Muted orange-yellow (#E07A5F), inspired by Picasso's later works, for selective highlights and interactive elements.
- Headline font: 'Playfair', a modern serif with high contrast, for an elegant feel.
- Body font: 'PT Sans', a humanist sans-serif, for readability and modern flair.
- Custom-designed icons based on Hardik's initials. Symbol/logo re-imagined to match a cubist-style aesthetic
- Re-imagined layouts that breaks typical structures to offer a unique user experience with locked background photos.
- Background animation consisting of several blended photos from one event displayed in an animated-GIF-like experience, only progressing as the user scrolls.
- Individual photographs should include animations, such as zooming or a cubist-like reshuffling of the image as the user scrolls.
